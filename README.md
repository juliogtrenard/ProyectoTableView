# Titulo del proyecto

ProyectoTableView

## Descripción

Aplicación JavaFX que muestra una tabla de personas con funcionalidades para añadir, eliminar y restaurar filas.

## Primeros pasos

### Cómo usar

* Clona el repositorio en algún IDE y ejecuta el programa.
